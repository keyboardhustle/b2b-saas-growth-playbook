# B2B SaaS Growth Playbook

A GTM and growth operating system for B2B SaaS marketers. This is not a collection of tactics — it’s a system for thinking, deciding, and executing. Use it to build a repeatable growth engine from scratch, or to diagnose and fix a broken one.

## Philosophy

**Marketing in B2B SaaS fails in predictable ways:**

1. No ICP definition → campaigns target everyone → leads are cheap but pipeline is thin.
2. No messaging grid → each channel says something different → brand is weak and Sales ignores marketing content.
3. Tactics without a system → busy work with no compounding → growth stalls after initial wins.
4. No experiment discipline → decisions based on gut → nothing is learned, nothing improves.

This playbook solves each of those in order: start with ICP, build messaging, then run channels, then run experiments.

## How to Use This

1. **Start with [`icp-framework.md`](docs/icp-framework.md).** Fill it in for your company. Nothing else works without it.
2. **Fill in [`messaging-grid.md`](docs/messaging-grid.md)** using your ICP output. Every channel should draw from this grid.
3. **Use [`experiment-backlog-template.md`](docs/experiment-backlog-template.md)** to log, prioritise, and track all growth experiments. Run one big experiment per channel per month.
4. **Use [`channel-playbooks.md`](docs/channel-playbooks.md)** when launching or fixing a specific channel.
5. **Use [`product-launch-checklist.md`](docs/product-launch-checklist.md)** for every product or feature launch.
